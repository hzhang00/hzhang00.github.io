Why?

Our approach to choosing an idea for our project was to choose something that could potentially change a problem and be helpful. We chose this project because we thought gun violence is a pressing problem in society today, and we wanted to use AI to provide a way to help people affected by it. With gun violence being seen throughout the USA today, we wanted to find correlations between different aspects of the scene, present them in an easy manner through visualizations, and also help police find a way to efficiently have the ability to react to an event like this in the future.

Hypothesis

We want to be able to predict the number of people that are injured during a shooting based on various features like the type of gun, where it is, and the gender and age group of the participants and be able to relay this information to the hospital so that they can prepare and get ready for the incident.

MVP

- Cool visualizations to understand the data at hand
- Trying to predict how many people would get injured in an incident
- Front end website to present this.

Classification or Regression

This is a regression problem; the difference between classification and regression is if it can be predicted/classified into a group, it is classification; if the model can only predict, it is regression; since what were are trying to figure out cannot be classified into a limited amount of numbers, it can be any amount from 1 to infinity making it a regression model.



Feature and Target Variables

Some features we had in our dataset were the number killed, participant descriptions like gender and age, location, and guns. The output or target variable was a prediction of the number of injured so that in a real-life scenario, proper equipment and assistance can be provided.

Linear regression

Linear regression models are used to find a linear correlation between two or more variables by drawing the line of best fit.

We chose this model because we wanted to find correlations between different features. We used this model as a beginner model to learn how it works so that the other more complicated models would be easier to learn.



We got a mean squared error of about 0.393 from this model.

Gradient Boosting

Gradient boosting is an ensemble model that uses a group of weak prediction models like decision trees and combines data from predictions and previous models to make it more accurate.

We chose to make a gradient boosting model because we wanted a model that was accurate. We started the model by getting the shape of the data and cleaning it and then doing an 80-20 split for the training-testing datasets.

We obtained a mean squared error of about 0.347.

Random Forest

Random Forest is an ensemble model that uses decision trees that filter each data point into different branches. From there, they choose the average of the end branches as the prediction.

We created this random forest regression model because it worked best with many features, like our data. We did hyperparameter tuning to further enhance our model.

We got a result of about 0.346 for our mean squared error from this model.
